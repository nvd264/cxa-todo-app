# cxa-todo-app
CXA - React Native - Todo

# Setup

## Clone source code
```javascript
git clone https://github.com/nvd264/cxa-todo-app.git
```

# Install
```javascript
cd CXAToDoApp && yarn install
```

# Run on iOs
```javascript
react-native run-ios
```
# Run on Android
```javascript
react-native run-android
```
